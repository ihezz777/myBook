Introduction

在系统里，纯函数与非纯函数相比，在可测试性、可维护性、可移植性、并行计算和可扩展性方面都有着巨大的优势。

相同的输入总是会返回相同的输出
var xs = [1,2,3,4,5];

// 纯的
xs.slice(0,3); // [1,2,3]
xs.slice(0,3); // [1,2,3]
xs.slice(0,3); // [1,2,3]

// 不纯的
xs.splice(0,3); // [1,2,3]
xs.splice(0,3); // [4,5]
xs.splice(0,3); // []
不依赖于外部状态
// 纯的
var checkAge = function(age) {
  var minimum = 21;
  return age >= minimum;
};

// 不纯的
var minimum = 21;
var checkAge = function(age) {
  return age >= minimum;
};
